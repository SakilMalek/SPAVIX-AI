## Project Overview

**SPAVIX-Vision** is an intelligent interior design platform that uses advanced AI to transform room images into professionally styled spaces. Built during my college internship, this project leverages Google's Gemini 2.5 Flash AI to provide instant room redesigns with photorealistic results.

### Core Technology
- **2-Stage AI Pipeline**: First detects room type (bathroom, kitchen, bedroom, etc.), then applies style transformations while respecting room-specific constraints
- **Smart Room Detection**: Prevents illogical outputs (e.g., no toilets in living rooms, no sofas in bathrooms)
- **Photorealistic Rendering**: Maintains original room geometry, camera angles, and object placement while changing colors, finishes, and decor

### Tech Stack
- **Frontend**: React + TypeScript + TailwindCSS + Radix UI
- **Backend**: Node.js + Express + PostgreSQL
- **AI**: Google Gemini 2.5 Flash (image generation) + Gemini Vision (room detection)
- **Auth**: JWT with refresh tokens + Google OAuth 2.0
- **Payments**: Stripe (planned)
